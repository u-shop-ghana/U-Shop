# U-Shop — Campus Tech Marketplace

> Ghana's trusted student tech marketplace with escrow-protected payments, verified sellers, and standardised condition grading.

## 📂 Project Structure

| Folder | Purpose |
|--------|---------|
| `docs/` | All documentation (product, technical, brand, legal, operations) |
| `design/` | UI/UX design files, wireframes, prototypes, research |
| `development/` | All source code (frontend, backend, mobile, database) |
| `business/` | Business planning, financials, marketing, partnerships |
| `assets/` | Brand assets, logos, images, icons, fonts, marketing materials |
| `testing/` | QA — unit, integration, e2e, performance, security tests |
| `deployment/` | DevOps — Docker, CI/CD, Terraform, Nginx, monitoring |
| `packages/` | Shared packages (UI components, API client, utils, types) |
| `documents/` | Original specification documents (PRD, architecture, etc.) |

See [CONTRIBUTING.md](./CONTRIBUTING.md) for the complete developer onboarding guide, setup instructions, and environment variable requirements.

## 📋 Tech Stack

- **Frontend:** Next.js 16 (App Router) + Tailwind CSS
- **Backend:** Express.js + Prisma ORM
- **Database:** PostgreSQL (Supabase)
- **Auth:** Supabase Auth
- **Payments:** Paystack (Cards, MoMo, Vodafone Cash)
- **Cache:** Upstash Redis
- **Email:** Resend
- **Monitoring:** Sentry + PostHog
- **Deployment:** Vercel (Web) + Railway (API)
